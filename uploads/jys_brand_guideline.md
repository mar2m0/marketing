## 1. 브랜드 철학

### Brand Essence
> **"신뢰를 쌓는 크리에이티브"**  
> JYS 마케팅은 화려한 기교보다 진정성 있는 이야기로 브랜드와 고객을 잇습니다.

### Core Values

| 가치 | 정의 |
|------|------|
| **신뢰 (Trust)** | 데이터와 경험에 기반한 일관된 결과물로 파트너의 믿음을 얻는다 |
| **전문성 (Expertise)** | 마케팅 트렌드와 AI 기술을 실무에 접목하는 깊이 있는 역량 |
| **진정성 (Authenticity)** | 클라이언트 브랜드의 본질을 먼저 이해하고, 있는 그대로 전달한다 |
| **성장 (Growth)** | 단기 성과보다 브랜드 자산의 장기적 성장을 함께 설계한다 |

## 2. 브랜드 퍼스낼리티

JYS 마케팅은 다음의 성격을 가진 브랜드입니다.

```
"경험 많은 선배처럼 실질적인 조언을 건네되,
따뜻하고 편안한 태도로 곁에 있는 파트너"
```

### 퍼스낼리티 스펙트럼

| 속성 | ← | → |
|------|---|---|
| 전문적 | ●●●●○ | 친근한 |
| 차분한 | ●●●●○ | 활기찬 |
| 신뢰감 | ●●●●●| 혁신적 |
| 간결한 | ●●●○○ | 풍부한 |

### 브랜드가 되어야 하는 것 / 아닌 것

| ✅ JYS 마케팅은 | ❌ JYS 마케팅은 아닌 것 |
|---|---|
| 신뢰감 있는 전문가 | 과장되거나 자극적인 브랜드 |
| 따뜻하고 진정성 있는 파트너 | 차갑고 거리감 있는 에이전시 |
| 실질적인 결과를 만드는 실행자 | 이론만 많은 컨설턴트 |
| 클라이언트 관점에서 생각하는 동반자 | 일방적으로 주도하는 공급자 |

## 3. 컬러 시스템

어스톤(Earth Tone) 기반의 따뜻하고 신뢰감 있는 컬러 팔레트를 사용합니다.

### Primary Colors (주 색상)

| 이름 | HEX | RGB | 용도 |
|------|-----|-----|------|
| **JYS Clay** | `#8B5E3C` | 139, 94, 60 | 메인 브랜드 컬러, 로고, 주요 CTA |
| **JYS Sand** | `#D4A97A` | 212, 169, 122 | 서브 강조, 아이콘, 포인트 컬러 |
| **JYS Cream** | `#F5EDE0` | 245, 237, 224 | 배경, 여백, 카드 배경 |

### Secondary Colors (보조 색상)

| 이름 | HEX | RGB | 용도 |
|------|-----|-----|------|
| **JYS Stone** | `#6B6560` | 107, 101, 96 | 본문 텍스트, 보조 설명 |
| **JYS Moss** | `#7A8C6E` | 122, 140, 110 | 긍정·성공 메시지, 포인트 |
| **JYS Ivory** | `#FDFAF6` | 253, 250, 246 | 전체 페이지 배경 |

### Neutral (뉴트럴)

| 이름 | HEX | 용도 |
|------|-----|------|
| **Dark** | `#2C2825` | 헤드라인, 강조 텍스트 |
| **Mid Gray** | `#A09890` | 비활성 요소, 캡션 |
| **Light** | `#EDE8E3` | 구분선, 테두리 |

### 컬러 사용 비율 가이드

```
Primary (Clay + Sand)  ██████░░░░  30%
Neutral (Cream + Ivory) ████████░░  50%
Secondary (Stone + Moss) ███░░░░░░░  15%
Dark / Text              █░░░░░░░░░   5%
```

> ⚠️ 채도가 높은 형광색, 차가운 계열의 블루·퍼플 계열은 브랜드 톤과 맞지 않아 사용하지 않습니다.

## 4. 타이포그래피

### 국문 서체

| 구분 | 서체 | 용도 |
|------|------|------|
| **헤드라인** | Pretendard Bold (700) | 제목, 핵심 메시지 |
| **서브헤드** | Pretendard SemiBold (600) | 소제목, 섹션 타이틀 |
| **본문** | Pretendard Regular (400) | 설명 텍스트, 바디 카피 |
| **캡션** | Pretendard Light (300) | 출처, 보조 설명 |

> 대체 서체: Noto Sans KR (무료 구글 폰트)

### 영문 서체

| 구분 | 서체 | 용도 |
|------|------|------|
| **헤드라인** | Playfair Display Bold | 영문 강조 제목 |
| **본문** | Inter Regular | 영문 바디 텍스트 |

### 타이포 사이즈 가이드 (SNS 기준)

| 요소 | 크기 | 비고 |
|------|------|------|
| 메인 헤드라인 | 48–64px | 카드뉴스 주 제목 |
| 서브 헤드라인 | 28–36px | 섹션 제목 |
| 본문 텍스트 | 18–22px | 읽기 편한 최소 크기 |
| 캡션 | 14–16px | 출처, 태그 |

### 타이포 원칙

- 한 디자인 내 서체 종류는 최대 **2종**으로 제한
- 자간(Letter Spacing)은 헤드라인 **-0.5%**, 본문 **0%** 기본 적용
- 행간(Line Height)은 본문 기준 **1.6–1.8배** 유지
- 텍스트 정렬은 **좌측 정렬** 또는 **중앙 정렬** 원칙 (우측 정렬 지양)

### 로고 유형

| 유형 | 사용 환경 |
|------|-----------|
| **Full Logo** (워드마크 + 심볼) | 웹사이트, 제안서, 공식 문서 |
| **Wordmark** (텍스트만) | SNS 프로필, 배너, 협소한 공간 |
| **Symbol** (심볼만) | 파비콘, 앱 아이콘, 워터마크 |

### 로고 컬러 버전

| 버전 | 배경 | 사용 상황 |
|------|------|-----------|
| **Primary** (Clay on Cream) | 밝은 배경 | 기본 사용 버전 |
| **Reverse** (Cream on Clay) | 어두운 배경 | 다크 배경 적용 시 |
| **Mono Black** | 흰 배경 | 흑백 인쇄물 |
| **Mono White** | 어두운 배경 | 흑백 반전 인쇄물 |

### 최소 여백 (Clear Space)

```
로고 높이를 X라고 할 때,

┌──────────────────────┐

└──────────────────────┘
```

## 6. 톤 앤 매너

### 브랜드 보이스 원칙

JYS 마케팅의 모든 커뮤니케이션은 다음 3가지 원칙을 따릅니다.

**① 명확하게 (Clear)**  
- 어려운 마케팅 용어보다 클라이언트가 이해하기 쉬운 언어 사용
- 핵심 메시지를 먼저, 부연 설명은 뒤에

**② 진정성 있게 (Authentic)**  
- 과장된 수식어나 근거 없는 최상급 표현 지양
- 실제 결과와 경험에 기반한 메시지

**③ 따뜻하게 (Warm)**  
- 딱딱한 공급자 언어가 아닌 동반자로서의 어조
- 클라이언트의 상황에 공감하는 표현

---

### 카피라이팅 예시

| 상황 | ❌ 하지 않는 표현 | ✅ JYS 마케팅의 표현 |
|------|-----------------|------------------|
| 서비스 소개 | "업계 최고의 마케팅 솔루션" | "귀사의 브랜드에 맞는 전략을 함께 설계합니다" |
| 결과 강조 | "매출 폭발 보장!" | "데이터 기반으로 꾸준한 성장을 만들어 갑니다" |
| 문의 유도 | "지금 당장 연락하세요!!!" | "편하게 먼저 이야기 나눠보세요" |
| 전문성 표현 | "수십 년 경력의 전문가 집단" | "현장 경험과 최신 트렌드를 함께 갖춘 팀" |

### 해시태그 사용 원칙

- SNS 게시물당 해시태그 **5–10개** 권장 (과도한 태그 지양)
- 브랜드 고유 태그: `#JYS마케팅` `#jysmarketing` 항상 포함
- 트렌드 태그는 콘텐츠 맥락과 관련 있을 때만 사용

## 7. 콘텐츠 비주얼 원칙

### 사진 & 이미지 스타일

| 항목 | 원칙 |
|------|------|
| **색감** | 따뜻한 화이트 밸런스, 과도한 채도 보정 금지 |
| **분위기** | 자연광 또는 자연광에 가까운 조명, 깔끔한 배경 |
| **인물** | 자연스러운 표정, 연출보다 진정성 있는 장면 |
| **오브젝트** | 어스톤 계열 소품 (나무, 종이, 도자기 질감) 활용 권장 |
| **필터** | 프리셋 통일 — 따뜻한 톤, 약한 페이드, 미세한 그레인 허용 |

### 카드뉴스 레이아웃 원칙

- 1장: 핵심 메시지 1개만 → **임팩트 강하게**
- 여백을 두려워하지 말 것 → **화이트스페이스 적극 활용**
- 텍스트는 전체 면적의 **40% 이내**로 제한
- 슬라이드당 읽는 시간 **3초 이내**를 목표로 구성

### SNS 채널별 비율

| 채널 | 권장 비율 | 비고 |
|------|-----------|------|
| 인스타그램 피드 | 1:1 또는 4:5 | 정사각형 우선 |
| 인스타그램 스토리·릴스 | 9:16 | 세로형 |
| 유튜브 썸네일 | 16:9 | 1280×720px 이상 |
| 블로그 대표이미지 | 16:9 또는 3:2 | |
| 네이버 포스트 | 1:1 또는 2:1 | |

## 8. 금지 사항

브랜드 일관성을 해치는 다음 행위는 모든 콘텐츠에서 금지합니다.

### 컬러

- ❌ 브랜드 지정 외 임의 컬러 조합 사용
- ❌ 형광·원색 계열 강조 컬러 사용
- ❌ 로고 컬러 임의 변경

### 타이포그래피

- ❌ 지정 서체 외 임의 폰트 혼용 (특히 고딕 계열 3종 이상 혼합)
- ❌ 텍스트에 그림자, 외곽선 등 과도한 효과 적용
- ❌ 전체 대문자(ALL CAPS) 남용

### 로고

- ❌ 로고 비율 변형 (찌그러뜨리거나 늘이기)
- ❌ 로고 위에 다른 요소 겹치기
- ❌ 배경과 대비가 낮아 식별 불가한 컬러 조합
- ❌ 로고 회전 또는 기울기 적용

### 커뮤니케이션

- ❌ 근거 없는 최상급 표현 ("업계 최고", "국내 1위")
- ❌ 과도한 느낌표·이모지 남용
- ❌ 경쟁사 비방 또는 비교 우위 직접 언급
- ❌ 클라이언트 정보 동의 없이 공개
